# Figures and Plots

This directory contains generated figures from training, evaluation, and real drone testing.

## Current Results

### Root Level Comparison Plots
- `comparison_*.png` - Controller comparisons for each trajectory type
- `comparison_*_robust.png` - Comparisons with domain randomization
- `evaluation_summary*.csv` - Numerical performance metrics

### Phase 1: Simulation Results
See [phase1_simulation/README.md](phase1_simulation/README.md) for details.
- Perturbation test plots
- Controller comparison visualizations
- Tracking error analysis

### Phase 2: Real Drone Results
See [phase2_real_drone/README.md](phase2_real_drone/README.md) for details.
- Autonomous flight analysis
- Wind perturbation test results
- Sim-to-real transfer validation

## Generating Figures

Figures are generated by analysis scripts:

```bash
# Simulation perturbation tests
python scripts/phase1_simulation/test_simulation_perturbations.py
python scripts/phase1_simulation/analyze_simulation_results.py

# Real drone analysis
python scripts/phase2_real_drone/analyze_perturbation_tests.py
python scripts/shared/analyze_hybrid_models.py
```
